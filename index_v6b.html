Panduan Setup & Referensi
1
Install dependensi Python (sekali saja)
pip install openpyxl watchdog
2
Edit EXCEL_FILE di watcher.py
Ubah path ke lokasi file Excel yang tersync OneDrive. Sesuaikan SHEET_MAP jika nama sheet berbeda.
3
Jalankan watcher (tiap masuk kerja)
python watcher.py
4
Hosting di LAN kantor
python -m http.server 3000
Akses dari komputer lain: http://[IP-PC-SERVER]:3000
